- JSX 파일 : .js 파일에서 쓰는 html 대용품
    - 클래스 추가시 `className = { class }`으로 작성 필요 (예약어 방지)
    - `let varName = text;` 으로 선언한 변수의 내용을 JSX에 삽입하는 경우
        - `{ varName }`를 넣어 대입 가능
    - `return ()` 안에는 병렬로 태그 2개 이상 기입 금지
        - 하나의 태그 내부에서 계층적으로 반환해야 함
        - 병렬 사용이 필요하다면 최상위 계층에 빈 `<></>` 태그로 감싸기
    - `html`과 다르게 JSX에서 **속성값 전달**은 `attribute = {value}`로 전달 
<br>

- useState 사용법
    1. `import {useState} from 'react'`
    2. `const [state, setState] = useState("initValue");`
        - `state`는 변수명
        - `setState`는 변수 설정
        - `initValue`에는 배열 및 객체도 넣을 수 있음
        - Destructuring 문법 : 배열 변수 선언에 배열을 할당할 수 있음
            - ex : `const [a, b] = [1, 2]` $\rightarrow$ `a = 1, b = 2`
    3. `state` 변수의 값이 변경되면 `html`이 자동으로 재 렌더링
        - 자주 변경될 것 같은 `html`부분은 `state`로 설정하는 것이 유용
    4. `setState`를 활용해야 `state`를 변경하고 재 렌더링 가능
        - 배열이나 객체의 부분 데이터 수정은 copy 변수를 만들어 복사한 뒤 의도에 맞게 
        변경한 복사본을 출력하는 형태로 사용하는 것이 관습 
        - `[...state]`로 복사한 뒤 변경해야 적용됨

- 컴포넌트 문법
    - 생성 방법
        1. `function App() { }` 외부에 새로운 컴포넌트를 만들기 위한 
        `function componentName() { }` 생성 
            - `const componentName = () => { }`로도 사용 가능
        2. `return ()` 안에 컴포넌트로 사용할 html 담기
        3. `function App() { }` 내부에서 `<componentName></componentName>`을 사용
    - 사용 예시
        1. 반복적인 html 코드 축약에 활용
        2. 큰 페이지 
        3. 자주 변경되는 html 코드
