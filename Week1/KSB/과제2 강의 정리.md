### 과제 2 왕 초보 자바스크립트 정리
- 변수
    - 선언 
        - 예약어 불가
        - let(변수), const(상수) 사용
            - 일단 모든 변수는 const로, 이후 변하는건 let으로 변경
            - 상수는 가급적 대문자로 표현
<br>

- 자료형
    - 템플릿 리터럴 활용
    ```JS 
        const name = "KSB";
        const msg = `My name is ${name}`;
        console.log(msg);
    ```
    - NAN : Not a Number 
    - Undefined : 값이 할당되지 않음
    - null : 존재하지 않음
    - `console.log(name + "는" + age + "살");`로 + 연결 가능
<br>

- 다이얼로그
    - `alert("text")` : 알림창 ("확인"만 있음)
    - `prompt("text", "placeholder")` : 입력창과  ("값 전달 확인과 취소" 둘 다 있음)
        - 아무것도 없으면 null
    - `confirm()` : 알림창 ("확인과 취소"둘 다 있음 / true or flase 반환)
<br>

- (명시적) 형변환 
    - String("input") : 문자열로 변경 
    - Number("input") : 숫자로 변경
    - Boolean("input") : 0 -> 빈문자열, null, undefined, NaN / 나머지는 모두 1
    - toString() : 문자열로 변경하는데 null, undefined 불가능 / 진법 변환 가능
    - parseInt() : 문자열 앞부분의 숫자 부분을 Int로 반환 (나머지 문자는 삭제)
        - parseFloat()도 가능
        - 숫자로 무조건 시작해야함 (아닌경우 NaN 반환)
    
<br>

- 비교 연산자 / 조건문
    - `==` 과 `===`의 차이
        - 동등연산자 `==`은 출력되는 값이 같으면 true
        - 일치연산자 `===`은 출력되는 값과 typeof까지 같으면 true
    - if / else if / else 로 조건 분기 가능
        - `if (condition) {context} else if (condition) {context} else {context}`
    - switch (조건 비교 : 일치연산자)
        ```JS
            const type = 0; 
            switch() {
                case 0 :
                    "result1"; break;
                case 1 :
                    "result2"; break;
                case 2 :
                    "result3"; break;
                default :
                    "default case";
            }
        ```
<br>

- 논리 연산자
    - || : OR 연산자
    - && : AND 연산자
    - ! : NOT 연산자
<br>

- 반복문
    - for / while
    ```JS
        const range = 10;
        for (let i = 0; i < range; i++) {
            `context`
        }

        let i = 0;
        while (i < range) {
            `context`
            i++;
        }

        // do 문장 1회 실행 후 반복
        i = 0;
        do {
            `context`
            i++
        } while (`condition`)
    ``` 
<br>

- 함수
    - 함수 선언문 : `function name( [input] ) { "context" [return "output"] }`
    - 함수 표현식 : `let name = function([input]) {"context" [return "output"]}`으로 쓰고
    `name()`으로 사용 가능
        - 선언 위치 주의
    - 화살표 함수 : `let name = ([input]) => {"context" [return "output"]}`
    - 지역변수와 전역변수 구분 및 사용에 유의
    - 함수는 하나씩 작성 및 적절한 이름 부여 필요
<br>

- 객체
```JS
    // INFO 객체에서 없는 프로퍼티를 검색하면 undefined 반환
    const INFO = {
        name : "KSB",
        age : 23,
        other : other               // other로 축약 가능
        sayHello : function () {
            console.log(this.name)  // this 활용시 화살표 함수는 유의해서 활용
        }
    };

    // 새로운 객체 생성 함수 활용
    function makeObj(name, age, other) {
        return {
            name, age, other : "student"
        }
    }

    // 접근
    INFO.name;              // "KSB"
    INFO["age"];           // 23

    // 추가
    INFO.gender = "male";
    INFO["hairColor"] = "black";

    // 삭제
    delete INFO.hairColor;

    // in 연산자
    console.log(("school" in INFO)); // false 출력 (기준 : 키가 있는지 없는지)

    // for ... in (객체 내 전체 출력)
    for (let x in INFO) {
        console.log(INFO[x]);
    }
```
<br>

- 배열
    - 배열 내부에는 일반 자료형, 객체, 함수 등 제한이 거의 없음
    - 내부 메서드
        - .length : 배열 노드 개수 반환
        - .push("oneData") : 마지막에 추가
        - .pop() : 마지막 제거
        - .unshift("FewData") : 처음에 여러개 추가
        - .shift("") : 처음에 여러개 제거
    - for ... of 형태로 반복문 사용
<br>

### 과제 2 자바스크립트 중급 강좌 정리
- TDZ 예시
    ```JS
        let age = 23;
        function showAge() {
            console.log(age);   // 단일 존재시 전역의 age 참조

            let age = 20;       // 호이스팅이 아닌 TDZ 문제 발생
        }
    ```
<br>

- 생성자 함수
    ```JS
        function User(name, age) {
            // this = {};
            this.name = name;
            this.age = age;
            this.sayName = function() {
                console.log(`name = ${name} / age = ${age}`);
            }
        }

        const User1 = new User("KSB", 23);  // 생성자로 객체 생성에는 new 필요
    ```
<br>

- Computed property
    ```JS
        const login = ({ID, address, password}) => {
            const loginData = {
                // 대괄호로 묶어 키를 만들고
                // 내부에서 여러 가지의 조합 연산 가능
                [`${ID}@${address}`]: password 
            };
        };
    ```
<br>

- 객체 메서드
    - Object.assign({}, objName) : 객체를 복제
    - .keys(objName) : 객체의 키를 배열로 반환
    - .values(objName) : 객체의 값을 배열로 반환
    - .entries(objName) : [키, 값]을 배열로 반환
    - .fromEntries(objName) : [키, 값]을 객체로 반환
<br>

- 심볼 자료형
    - `const a = Symbol(); const b = Symbol();`에서
        - type은 같지만 `a === b;` 의 결과는 `false`
    - 객체 메서드에도 출력되지 않고 반복문에서도 출력되지 않음
    - 유일한 프로퍼티를 추가할 때 사용
<br>

- 문자열 메서드
    - 전제 대문자 변환 : `toUpperCase()`
    - 전체 소문자 변환 : `toLowerCase()` 
    - 문자열 일부 찾기 
        - 시작 index 기준 : `indexOf("text")` 
        - 문장 전체 내부 기준 : `includes("text")`
    - 문자열 자르기 : `slice(start, [end])` / `end`가 없으면 문자열 끝까지
    - 공백 제거 : `trim()`
    - 문자열 반복 : `repeat()`
<br>

- 배열 메서드
    - `.splice(start, count[, add])` : 특정 요소 지우고 동시에 반환
    - `.slice(start, end)` : 특정 요소 부분만 반환 (빈 경우 전체 반환)
    - `.concat(a, b)` : 두 배열을 이어 새 배열 반환
    - `.forEach((item, index[, arr]) => {})` : 배열 전용 반복
    - `.find((`condition`) => {})` : 조건에 맞는 값 개수 반환
        - `.findIndex()` 로 인덱스 반환도 가능
        - `.filter()` 로 해당 배열 반환도 가능
    - `.join("text")` : 배열을 연결해 문자열로 반환
    - `.split("text")` : "text"를 기준으로 문자열을 분리해 배열로 반환
    - `.sort((a, b) => { return a - b })` : 정렬 (숫자, 알파벳) / 
        - 또는 `.sortBy(arr)` 사용
    - `.reduce((prev, cur) => { return prev + cur }[, inital value])` 
    : 누적된 배열의 총 합 + 현재 값
<br>

- 구조분해할당
    - 객체 활용 (2개의 구문은 같음)
        - `let user = {name: "KSB", age: 30};`
        - `let {name, age} = user;`
<br>

- 나머지 매개변수
    - 정해지지 않은 여러개의 매개변수 활용 가능
    - 가장 마지막에 있는 매개변수여야 함
    ```JS
        function add (...numbers) {
            let result = 0;
            numbers.forEach((num) => (result += num));
        }
    ```
<br>

- 클로저 (?)
<br>    

- `.call(this 대상, [a, b, ...])` : this 대상을 특정값으로 나열해 전달
- `.apply(this 대상, arr)` : this 대상을 배열로 전달
- `.bind(this 대상)` : 함수를 this 대상으로 고정하여 사용
<br>

- 상속, 프로토타입
    - `객체1.__proto__ = 객체2` : 객체 1에 프로토타입 객체 2를 상속
<br>

- 클래스
    - 생성자의 특성을 이용하여 잘못된 선언의 경우를 방지할 수 있음
    - extends 상속을 통해 `__proto__`를 추가할 수 있음
    - 생성자 오버라이딩 활용 시 super() 생성자 확인 필요
<br>

- 프로미스
    - .catch - .finally 로 에러 처리 필요
    - 프로미스 반환 함수를 만들고 이후 .then 으로 연결하여 사용하는 방식
    - 화살표 함수 사용 필요
    - `Promise.all()`을 활용하여 모두 실행되고 출력하게 하는 방식도 있음
- async, await 
    - try - catch - finally 블록으로 에러 처리 필요
    - await를 통해 우선 순위를 결정하여 연속으로 실행 가능
<br>

- Generator
    - `function*` 로 선언
    - `yield 'n';`를 활용하여 중간에 얼마나 쉬었다가 다시 재개할지 결정
        - 외부 입력을 받아 결정할 수도 있음
    - 필요한 순간까지 계산을 미루기 위해서 사용
    - 이후 `next()` 해주면 멈췄던 부분부더 이어서 실행
